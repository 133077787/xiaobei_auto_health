# 第二种使用方法


首先在 config.py 中写入你的身份证号和密码（默认为身份证号后六位）

注意不要去掉双引号

```python
username = "100000199801020022"
password = "262428"

```

在完成后，它应该像上面这样

# 运行

```bash
pip3 install requests

python3 main.py

```
